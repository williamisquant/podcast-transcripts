## 1. 總經 / 盤勢

### 核心觀點
謝孟恭認為目前盤勢非常強，美股與台股都在走強勢多頭，拉回反而偏向買點，而不是先預設崩盤。

### 重點
- 美股近期大漲，台股也非常強。
- 他認為市場上檔空間可能超出很多人想像。
- 以台股來看，因為台積電權重很大，若台積電維持強勢，指數有機會往更高位置推進。
- 現在不是他會主張到處恐嚇「快崩了」的時間點；相反地，只要沒有重大利空，強勢多頭中的回檔通常會有人接。

### 操作心法
- 不預設高點
- 在好行情裡盡量找還有 upside 的標的
- 不要因為漲多就本能看空
- 派對會結束，但在結束前要先把握行情

## 2. 產業 / 供應鏈

### 廠務股
- 台積電法說後，廠務股開始有表態。
- 他過去持有不少廠務股，但因為漲得慢，有減碼去換更快的標的。
- 廠務股仍偏便宜區，只是不一定是最有動能的族群。

### 被動元件
- 國巨法說偏保守，但他不認為這代表產業差。
- 國際被動元件相關標的表現很強，台灣這邊節奏相對慢。
- 市場上已看到漲價函、稼動率上升、電容 / 電感偏缺，電阻也開始出現漲價訊號。
- 他用記憶體經驗類比：一旦稼動率進一步逼近滿載，供需可能突然轉緊。

### 記憶體
- 記憶體價格仍強，但台股相關股票相對美股有點疲弱。
- 這不一定是基本面轉差，更像是資金已上車、籌碼變重，市場在找下一個缺貨 / 漲價題材。

### CPU 供應鏈
- 他認為市場可能低估 CPU 供應鏈的催化與缺貨風險。
- CPU 缺貨可能比記憶體更嚴重：記憶體是貴但通常買得到，CPU 則可能想買都買不到。
- 若後續大量建置 General Purpose Server / AI Server，CPU 與相關零組件、測試、供應鏈都可能被帶動。
- 他把 CPU 形容成「房間裡的大象」：大家知道 Agentic AI 會用到 CPU，但市場可能還沒有充分反映影響有多大。

## 3. 個股 / 類股

### 台積電
- 依然是盤勢核心。
- 法說內容不差，資本支出雖未上修，但維持在高檔區間，代表信心仍在。
- 他認為未來仍有可能出現進一步上修。

### 國巨
- 被動元件代表股。
- 法說偏保守，但不代表產業趨勢結束。

### 國際觀察名單
- Vishay / VPG / Eton 被拿來對照國際被動元件供應鏈的強勢表現。

### CPU 測試設備相關
他在 QA 中提到自己偏好的幾個名字：
- 均華
- 致茂
- 萬潤

他的意思是這條線早就注意很久，只是因為股價漲多，不想在高檔一直公開講。

## 4. AI / 軟體與 token 論述

### Uber 案例
- Uber 開年沒多久就把 AI token 額度用光。
- 很多人會視為 AI 極度 bullish，但他更在意的是：燒這些 token 後，到底換到了什麼實際價值？

### 核心質疑
他現在看 AI 應用，不是只看：
- 有沒有用 AI
- 用了多少 token

而是看：
- 產品在哪裡
- shipment 在哪裡
- 財務數字有沒有改善
- 毛利 / 淨利 / 效率有沒有提升

### 對軟體公司的估值思路
- 市場未來會更重視 result，而不只是 token maxing。
- 不是所有號稱在用 AI 的公司都會成功，很多可能只是多花錢、沒創造足夠價值。

### OpenAI / Anthropic
- 他提到 OpenAI 甚至會去質疑 Anthropic 的營收認列方式。
- 這代表市場也會開始更現實地檢驗 AI 敘事。

## 5. 人生觀 / 價值觀

### 關於陪伴、死亡與生命
- 節目前段由一位太太寫給已過世丈夫的訊息切入。
- 他延伸談到，自己現在比較願意相信神靈、能量，甚至來世的可能性。
- 不是因為有科學證據，而是人會希望與重要的人再相見。

### 健康比數字重要
- 他用很鮮明的比喻：財富、工作、成就都像前面的數字，健康是最後那個乘數。
- 如果健康是 100%，前面的數字才成立；如果健康是 0%，乘起來就是 0。

### 什麼叫成功
- 早期的他會認為有錢就是某種程度的成功。
- 現在他認為成功絕對不是只有有錢，而是：
  - 身體健康
  - 家人小孩過得不錯
  - 能過自己想過的生活
  - 對自己有交代
- 成功的定義會隨年紀改變，不要照單全收社會給的標準答案。

- 不用學別人怎麼花錢，每個人值得花錢的地方本來就不同。

## 一句話總結

在強勢多頭與供應鏈催化延伸的背景下，謝孟恭一邊看好 CPU / 被動元件 / 廠務等後續機會，一邊提醒大家：AI 不只要會燒 token，更要產出結果；而比投資績效更根本的，是健康、家人與你怎麼定義自己的成功。
