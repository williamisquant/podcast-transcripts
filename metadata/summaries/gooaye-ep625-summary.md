## 1. 新年的開始沒有很盛大，但很真實：在家打手遊、抽卡、順便接受自己的人生階段變了

本集開場先由 Backbone 業配帶到人體工學家具，不過謝孟恭很快就切回自己的跨年夜。他說這次跨年幾乎是在家跟朋友一起抽手遊角色度過，抽著抽著時間就過了，跟大家說一聲新年快樂後，心裡反而冒出一種複雜感：一方面覺得這樣其實很快樂，一方面又有點覺得自己是不是太宅、太平淡了。

但他很快也接受這就是人生階段的轉變。以前會衝 101 看煙火、會在意跨年儀式感，現在光想到人擠人、擠進去還要擠出來，就覺得累。再加上有小孩之後，像西班牙吃葡萄這類跨年儀式，也會開始考慮安全與體力成本。這段表面上是生活閒聊，實際上很像整集的情緒底色：很多事情不是不有趣了，而是你開始更願意承認「舒服、平穩、可持續」本身就是一種價值。

## 2. 指數很強，但真正值得看的是 AI 高共識股開始鬆動

進入盤勢後，謝孟恭先很坦白地說，開年至今自己甚至沒有打贏大盤，代表這波上漲的廣度與強度都相當驚人。對很多滿手記憶體、面板等族群的人來說，可能績效早就遠遠領先。這種開場其實很有代表性，因為他不是先吹自己，而是先承認市場就是有時候比你更瘋、更快。

但真正重要的不是指數多強，而是盤面結構開始出現變化。他特別點出，原本市場最共識、法人最愛、幾乎被視為 AI 核心配置的那些股票——像網通、散熱、高階材料與相關設備股——開始出現同步回檔。這不是單一公司的問題，而更像資金開始從原本最擁擠的主線裡抽出一部分。

謝孟恭的解讀不是立刻翻空，而是認為資金偏好正在重新排序。強勢主線可能還在，但不代表每一檔高共識股都會永遠是最舒服的地方。當市場開始覺得老故事太擠，就一定會有人去找新故事，或至少先從最擁擠的位置提款。

## 3. CES 之後，NVIDIA 真正在講的不是單顆晶片，而是 whole rack 的統治力

本集的核心產業重點，來自 NVIDIA 在 CES 與後續新架構所釋放的訊號。謝孟恭認為，老黃這次真正想強調的，不只是某顆新晶片效能更高，而是 NVIDIA 正在把自己的影響力從 compute 繼續往整個機櫃、整個系統、整個 solution 往外延伸。

他把關鍵字抓得很準：whole rack solution。也就是說，未來真正有價值的不只是 GPU 本身，還包括新的線材、新的 rack 設計、散熱方案、交換架構，以及整櫃層級的整合能力。從這個角度看，硬體並沒有像某些人期待的那樣「很快就告一段落」，反而可能因為系統層級越做越深，而持續有新的硬體延伸故事跑出來。

這個判斷很重要，因為它直接修正了很多人常有的簡化想法：好像 AI 基建走一段後，就只剩軟體與應用。謝孟恭反而認為，只要系統層的效能還能繼續往上推，硬體就不會那麼快結束。它只是從單點零件，變成更完整、更高層級的整體方案。

## 4. ICMS 與 storage 新敘事：下一波不是只有算力，而是 context memory 怎麼被安放

本集最有技術含量的一段，是他拆解 NVIDIA 提出的 ICMS。由於 clean transcript 來自 ASR，這個縮寫的完整展開在逐字稿中有些辨識誤差，但從上下文可以明確看出，謝孟恭理解的重點是：NVIDIA 正在把手伸往 storage 與 inference 階段的 context memory 管理，試圖處理推論時大量 KV cache 與資料存取的壓力。

他的說法很清楚：如果未來由 BlueField DPU 驅動某種 storage node，讓資料在 rack 間與系統裡被更有效率地調度，那 NVIDIA 的掌控範圍就不只涵蓋 compute、switch、HBM，甚至會一路延伸到 storage server。也就是說，過去大家把儲存當成相對邊緣的配角，未來可能會因為推論與大上下文需求而變成新的兵家必爭之地。

這一段最值錢的地方，在於他不是把 ICMS 當新名詞照唸，而是直接抓住背後邏輯：NVIDIA 正在為下一輪的瓶頸先做布局。當市場還在盯著算力堆多少、HBM 多缺時，NVIDIA 已經開始往「推論時資料怎麼放、怎麼存、怎麼取」這件事走。對投資人來說，這就是新故事真正冒出的地方。

## 5. 群聯、PSLC 與 CES 現場照片：真正的研究優勢，往往來自你看懂一張別人隨手拍的圖

謝孟恭在節目中提到一張 CES 現場照片，有人拍到 NVIDIA 展示設備裡某塊板子上出現群聯控制器。這件事之所以有趣，是因為它不是來自正式新聞稿，而是來自現場照片裡被台灣人認出來的細節。對他來說，這就是研究裡很關鍵的一種優勢來源：別人看的是熱鬧，你看的是供應鏈線索。

他進一步推測，這塊板子上的 NAND 顆粒雖然可能不是群聯自己做，但透過控制器與整合，群聯已經有機會把東西真正做進 NVIDIA 的體系裡。若這條線後面不只是展示，而是進一步延伸到某種認證或特定 storage 架構，那群聯的故事就不只是傳統記憶體控制器，而是有機會升級成 AI storage 架構的受惠者。

他還提到 pseudo SLC 的做法，也就是用 TLC 顆粒模擬出類 SLC 的使用模式，以換取更高耐久度與更適合高頻寫入的特性。這一段很重要，因為它說明了 storage 不是隨便塞顆 SSD 進去就完事，而是為了配合特定推論與快取場景，要做出取捨與特殊配置。這也就是為什麼他會把這條線視為下一波值得追的重點。

## 6. 研究報告怎麼寫：不是把產業百科重抄一遍，而是要告訴我「新東西會怎麼改變舊世界」

QA 中有一位聽眾問到，自己做了第一份研究報告，想知道一份好的研究報告該長什麼樣。謝孟恭的回答非常有代表性，也很適合當成他的研究方法總整理。他認為，報告好不好，跟產業成熟度有關。若今天是一個自己不熟的產業，你把市佔、價格、供應鏈環節整理清楚，已經很有價值；但像記憶體這種他本來就很熟的產業，單純把市佔與報價重抄一遍，意義就不大。

對他來說，真正有價值的報告，是你能指出「新東西將如何改變既有供應鏈」。例如 NVIDIA 推 ICMS 之後，對記憶體與 storage 產業的影響是什麼？DDR4 轉 DDR5 的 redesign 會帶動哪些元件？哪些原本不起眼的 component 會因此需求暴增？這種資訊才是能創造研究優勢的部分。

換句話說，研究不是抄課本，而是找變化。你若只是把大家都知道的架構重新講一遍，頂多是把資料整理得漂亮；但若你能把新應用、新規格與既有供應鏈重新接上，那才是真正能幫助交易決策的東西。這其實也是整集最重要的方法論：市場永遠在找下一個變化點，而不是等你把舊世界背熟才給你獎勵。

## 7. 這集的真正底色：AI 基建還沒演完，只是舞台從算力開始往記憶體與 storage 擴散

如果把本集從跨年、盤勢、高共識股回檔、CES、whole rack、ICMS、群聯與研究報告的討論全部串起來，會發現它其實是在講同一件事：AI 基建的故事還沒有演完，但舞台正在轉移。

過去市場焦點主要放在 compute、HBM、封裝、散熱，如今謝孟恭認為，下一步已經開始往 context memory、storage、資料調度與更高層的系統整合移動。這意味著，舊主線不一定壞掉，但新的敘事會把資金與評價重新往更多供應鏈節點推開。

也因此，真正重要的不是短線鬼故事或某天跌了 10%，而是你有沒有跟上這個故事從哪裡延伸到哪裡。對他來說，市場裡最有價值的不是搶第一句口號，而是提早知道下一個瓶頸會長在哪，再去找誰會因為那個瓶頸而被重估。

## 一句話總結

這集從平淡跨年談到 CES、whole rack、ICMS 與群聯切入的 storage 故事，謝孟恭真正想講的是：AI 硬體主線還沒結束，它只是正從算力與記憶體，進一步往 storage 與整機架構深處擴張。
