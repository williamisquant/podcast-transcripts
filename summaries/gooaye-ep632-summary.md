# Gooaye 股癌 EP632 | ❄️ 摘要

## 提到的股票 / 公司
- Tesla
- 亞光
- 所羅門
- 原相

## 1. 歐洲生活讓他直接感受到地緣政治
- 一間俄羅斯商店改叫烏克蘭商店、Lisa 在某些場合刻意改講英文，這些細節都讓他很有感。
- 俄烏戰爭對歐洲不是新聞標題，而是會滲進日常生活與身份符號的東西。
- 這段生活觀察很小，但資訊量很大。

## 2. 在羅馬開手排車，幾乎是壓力測試
- 他對羅馬市區手排車的評價接近懲罰等級：窄路、坡道、貼車、亂停，一切都很靠北。
- 這段吐槽其實也在講：浪漫的文化想像，放進真實生活場景不一定成立。
- 很多規矩或執念，最後都要回到環境適不適合。

## 3. 特斯拉估值早就不是只看交車
- 他再次強調，若只看賣車，根本撐不起 Tesla 的估值。
- 真正支撐故事的是自駕、車隊化、Cybercab、Optimus 這些更遠的未來路徑。
- 所以縮邊車型、重配產線，不一定是利空，反而可能是公司在用行動押未來。

## 4. 自駕比很多人想的更近
- Transcript 裡自駕、road map、Optimus 都被明確提到。
- 他從公司語氣與動作裡感受到，自駕不再只是嘴巴講願景，而是開始影響資源配置。
- 如果公司內部真的把 unsupervised 自駕當主軸，市場估值邏輯就必須一起改。

## 5. 機器人從「太遠」變成「開始正式追蹤」
- 這集最大的轉折，是他對機器人態度變了。
- 過去嫌太遠，現在則因為 Tesla 把產線、capex、Optimus 一起講出來，開始把它納入研究範圍。
- 但他仍然很務實：台股目前多數機器人概念股純度不高，像亞光、所羅門、原相更適合先追蹤、別太快當成已可精算的主線。

## 一句話總結
這集最重要的不是 Tesla 再講了什麼夢，而是主委正式把機器人從「可以先不看」改成「要開始認真追」的題材。
