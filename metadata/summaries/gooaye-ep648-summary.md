## 1. 開場 / 健康與生活狀態

- 節目前段除了業配之外，謝孟恭花了不少篇幅談自己的健檢結果：膽固醇、Lp(a) 與血壓等數字讓他重新正視健康管理。
- 他坦言自己現在對身體的態度已和以前不同，因為想看到孩子長大，所以只要醫師認為需要調整，無論是吃藥、運動、減重或生活管理，他都願意配合。
- 這段也讓整集在市場話題之外，多了一層很鮮明的中年心境：投資再重要，最後還是要建立在健康還能讓你活著享受成果。

## 2. 總經 / 盤勢

### 美股
- 他認為美股的味道越來越像先前出現過的大回檔前夕，因此雖然逢跌仍有買一些，但整體上已經開始把現金水位慢慢拉高。
- 操作上不是全面看空，而是把弱勢破底股先提款，保留流動性，避免真的出現更大級別下殺時沒有子彈。
- 他的核心判斷是：美股現在仍然很凶險，如果後續反彈無法轉強，就要提防再一波大幅震盪。

### 台股
- 相比之下，台股顯得強很多，但他也不認為能完全脫離美股影響。
- 因此台股操作上會更偏向「強勢股若跌破關鍵結構也要出」，不是因為看壞產業，而是為了提早防守國際盤面帶來的連動風險。

## 3. 產業 / 供應鏈

### 光通與 CPU 仍是近期主線
- 他指出現在最火的仍然是光通，只是因為股價已經在右上角，自己反而刻意不在這個時間點大力鼓吹。
- 另一條他持續關注的是 CPU，因為推論端與通用運算需求仍在升高，這條線的供應鏈與封裝題材都可能持續發酵。
- 在操作上，他偏好把資金從弱勢破底標的移到仍在走強、或有新催化的主題。

### 玻璃基板 / TGV / GCS
- 這集最重的產業主題是玻璃基板。
- 他的核心看法很明確：Glass Core Substrate / TGV 是真的重要，但大規模量產時間點大約還要等到 2027 中後到 2028，現在談更多是在做中長期佈局與估值預演。
- 他提醒大家要把幾種玻璃相關技術拆開看：glass carrier、glass core substrate、glass interposer 不是同一件事，導入時間也不同，不能全部混成一團炒作。

### Intel、Google TPU、Broadcom 與台系供應鏈
- 他認為目前玻璃路線跑得比較前面的公司是 Intel，其次 Broadcom 也在積極測試與規劃。
- Google TPU v9 / 相關大型封裝需求，也讓 Intel 的 EMIB 路線再次被市場注意。
- 對台系供應鏈來說，這代表玻璃基板不是空穴來風，但要真的看到大幅貢獻，還是要等客戶導入、成本下降與量產節奏明朗。

### 技術與投資的時間差
- 他特別提醒：即便技術方向看對了，股價也不一定會照著技術成熟度線性反映。
- 市場往往會提早兩三年去買夢，像之前 CPU、光通就是這樣，所以玻璃題材也可能提早動，但不代表眼前的量產數字已經跟上。
- 因此這類題材要同時看技術 road map、客戶採用、價格下降速度與市場情緒，而不是只看單一故事。

## 4. 個股 / 公司觀察

- **Intel**：在玻璃基板與 EMIB 方案上是目前相對積極且具象的先行者。
- **Google TPU 線**：因大型封裝需求與 Intel 路線產生交集，是觀察玻璃基板進度的重要窗口。
- **Broadcom**：在日本與合作夥伴投入玻璃相關產線與測試，後續若交換器與光通產品導入，市場可能進一步強化這條敘事。
- **NVIDIA**：短期看起來還不是最早採用玻璃基板的那一方，若未來正式導入，反而可能代表題材已走到更成熟階段。

## 5. 人生觀 / 價值觀

- 他談到健康時，整個語氣非常直接：以前會覺得身體數字高一點沒差，現在則知道只要想陪小孩久一點，就沒有任何可以硬撐的空間。
- 這種心態其實也和投資很像：當你知道真正重要的是長期活著、長期留在場上，很多當下的任性就會自動縮小。

## 一句話總結

EP648 的重點是：在美股風險升高、台股相對強勢的背景下，謝孟恭一邊提高防守、一邊把目光放到光通、CPU 與玻璃基板這些中長期主線；而在市場之外，他也用自己的健康議題提醒大家，真正最值得長期投資的，還是讓自己能活著把好日子過完的身體。
