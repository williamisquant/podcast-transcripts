# Gooaye 股癌 EP625 | 🧸 摘要

## 提到的股票 / 公司
- NVIDIA
- 群聯

## 1. 開場很宅，但其實很真實
- 跨年幾乎是在家抽手遊角色度過，他一邊覺得很快樂，一邊也意識到自己的人生階段真的變了。
- 比起去外面擠，他現在更接受舒服、平穩、可持續本身就是一種價值。
- 這個情緒底色，跟後面談 AI 基建延伸其實很一致。

## 2. NVIDIA 真正想做的是 whole rack
- 他認為 CES 後最重要的訊號，不是某顆新晶片，而是 NVIDIA 往 whole rack solution 繼續推進。
- 也就是說，未來值錢的不只 GPU，而是線材、機櫃、散熱、交換、整櫃整合。
- 這直接修正「硬體快做完、只剩軟體」的簡化敘事。

## 3. ICMS / storage 是下一段新故事
- Transcript 裡明確提到 NVIDIA 的 ICMS。
- 主委抓到的核心不是縮寫本身，而是 NVIDIA 開始把手伸往 inference 階段的 context memory / storage 管理。
- 換句話說，下一波不只比算力，也比資料怎麼放、怎麼取、怎麼在系統裡調度。

## 4. 群聯 / PSLC 的重點在供應鏈位置
- 他提到 CES 現場圖裡疑似出現群聯控制器，並把這看成有機會卡進 NVIDIA storage 體系的線索。
- PSLC 之類做法的重要性在於：這不是一般 SSD 故事，而是為特定 AI 場景做的耐久度與效能配置。
- 他的研究優勢也很典型：別人看熱鬧，他看供應鏈細節。

## 5. 好研究報告不是重抄百科，而是找變化
- QA 裡他直接講研究方法：熟產業就不要重寫大家都知道的市佔與報價。
- 真正有價值的是，新規格、新架構會怎麼改寫既有供應鏈。
- 這也是本集最值得帶走的方法論。

## 一句話總結
這集真正想講的是：AI 硬體主線沒有結束，它只是從算力與 HBM，開始往 storage、context memory 與整櫃系統更深處延伸。
