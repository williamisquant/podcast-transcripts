# Gooaye 股癌 EP611 摘要

## 提到的股票 / 公司
- 旺宏
- Amazon
- Broadcom
- 台積電

## 指數沒那麼差，不代表大家都過得好
- 本集最重要的盤面觀察，是櫃買年初至今報酬轉負。
- 也就是說，如果你的持股偏中小型股，體感很可能和新聞上的 AI 大行情完全不同。
- 台股表面沒那麼差，很大一部分只是因為權值股、尤其台積電還撐著。

## 市場分裂與相對剝奪感
- 他把這件事延伸到更大的結構問題：少數權值與熱門 AI 題材撐住檯面，下面很多股票其實早就冷掉。
- 所以現在不是全面狂熱，而是很分裂的市場。
- 只看加權指數，很容易誤判自己其實正站在逆風區。

## 旺宏與 MLC NAND
- 產業面上，他特別提到旺宏的 MLC NAND。
- 若全球可供應者真的收斂到極少數，原本不起眼的利基產品就會被重新定價。
- 這種機會通常不在最熱門的 headline，而在供給退出、需求還在的角落。

## AWS Trainium / ASIC 供應鏈
- 另一條主線是 Amazon 的 AWS Trainium 與後續 ASIC 版本變化。
- 他提醒大家，自研晶片市場不是線性受惠：不同版本、不同規格、不同封裝需求，都可能重排受惠順序。
- Broadcom 等名字可以觀察，但不能只因一個 redesign 傳聞就下劇烈結論。

## 配置與人生觀
- 對投資配置，他再次強調別硬抄別人的答案。
- 別人的回檔耐受度、現金流、槓桿與睡眠品質，都不一定是你的版本。
- 談工作時也一樣：現在很多人追求的不只是養家，還有能否接受這種生活方式。

## 一句話總結
EP611 用櫃買轉負把台股「指數穩、體感差」的結構講清楚，也提醒大家真正的判斷不能只看表面熱度。
