# Gooaye 股癌 EP653 | 🕊️ 摘要

- 主持人：謝孟恭
- 集數：EP653 | 🕊️
- 日期：2026-04-15

## 1. 開場 / 生活話題

- 前段是 NordVPN 業配，重點放在旅遊比價、公共 Wi‑Fi 安全、切回台灣 IP、以及跨區使用情境。
- 謝孟恭延伸談自己的個性：他覺得自己屬於很有彈性、善變、能快速轉向的人，而這種特質在投資與工作裡未必是缺點。
- 他也分享自己最近對酒類文化、紅酒與酒櫃的興趣演變，作為一個「興趣快速投入、快速理解」的例子。

## 2. 總經 / 盤勢

- 他認為近期市場很強，美股明顯回神，台股也持續由強勢股往上推進。
- 在這種盤勢下，核心重點不是預設崩盤，而是把握多頭行情。
- 他也提到，在機會成本很高的盤裡，漲太慢的東西即使沒壞，也可能被調節，資金會往更強的標的移動。

## 3. AI / Token / 算力價格

- 他明確認為，長期來看 token 應該會越來越便宜，而不是越來越貴。
- 短期價格上升，主因是 AI 爆發速度快過產能擴張速度，造成供給跟不上需求。
- 他的核心判斷是：
  - 大家都在大量使用 AI 時，算力可能不足；
  - 但只要建置量慢慢跟上，價格仍會走向下降；
  - 長期更像 cloud 資源，會隨規模化而變便宜。

## 4. 產業 / 供應鏈

### 設備與 cleanroom
- 他提到美系設備廠直接指出：設備不是不能賣更多，而是 cleanroom 蓋得太慢，成為瓶頸。
- 這代表 AI 擴產不是單純晶片問題，而是整個基礎設施、無塵室、設備、建廠節奏一起受影響。

### 台積電 / 擴產節奏
- 他認為台積電原本對 AI 擴產相對保守，因為會擔心又是一輪半導體高點循環。
- 但在 AI 需求持續兩三年、且收益逐漸被驗證後，他覺得未來進一步上修產能規劃的機率很高。

### 聯發科 / Google TPU 合作線
- 這集有相當多篇幅在談聯發科與 Google TPU 相關合作。
- 核心意思是：聯發科在 AI ASIC / TPU 鏈上的參與程度，可能比很多人原先想像的更高。
- 他認為若後續量產與導入順利，聯發科在 AI 上的營收與 EPS 貢獻有機會顯著提高。

## 5. 個股 / 公司觀察

### 台積電
- 依然是台股核心，指數本質上很大程度受台積電影響。
- 只要台積電的中長期獲利預估沒有被明顯下修，指數上方空間就仍存在。

### 聯發科
- 這集是相對重要的觀察標的。
- 市場原本較擔心手機相關需求，但他認為新一波進來的資金，不只是看手機，而是看聯發科在 AI 裡的角色。
- 若 Google TPU 相關合作持續擴大，聯發科有機會被重新評價（rerate）。

### 其他延伸鏈
- 節目中也帶到與 TPU、封裝、材料、相關 IP / 元件有關的討論。
- 他的重點不是把每個零件講死，而是提醒大家：市場未來可能會沿著 TPU / AI infrastructure 繼續挖掘供應鏈機會。

## 6. 人生觀 / 價值觀

- 他把自己的「善變」「三分鐘熱度」「很快玩膩」重新理解成一種探索世界的能力。
- 他認為很多小時候被視為缺點的個性，長大後在工作與投資裡反而是優勢。
- 與其逼自己長期黏在單一興趣上，不如接受自己會快速探索、快速理解、快速轉移，只要知道自己在做什麼，這也可以是一種很好的活法。

## 一句話總結

EP653 的重點是：在 AI 基建與算力需求持續擴大的背景下，token 長期仍會走向更便宜、更普及；而聯發科 / Google TPU 這條線值得關注。同時，謝孟恭也用自己的個性經驗提醒大家，所謂善變與三分鐘熱度，不一定是缺點，也可能是面對快速變動世界的一種能力。
