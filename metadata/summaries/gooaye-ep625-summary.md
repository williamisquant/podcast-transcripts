## 開場 / 生活話題

- 謝孟恭先聊自己的跨年夜是在家跟朋友抽手遊角色、打遊戲度過，感嘆年紀與生活型態改變後，跨年儀式感也從往外跑變成在家過得舒服就好。
- 他也提到有小孩後，很多看似有趣的跨年習俗或活動都得重新思考安全與體力成本，開年心境相對平淡，但也更能接受這種生活節奏。

## 總經 / 盤勢

- 他認為 2026 年開年的台股非常火熱，自己坦言績效甚至還沒贏大盤，顯示這波上漲的廣度與強度都相當驚人。
- 不過他觀察到一個重要變化：過去市場最受歡迎、法人高度共識的 AI 核心持股，像網通、散熱、伺服器與高階材料族群，開始出現同步回檔，這不只是單一公司問題，更像資金偏好正在轉換。
- 對於市場上拿各種鬼故事去解釋下跌，他提醒不要在短線過度套基本面理由。很多消息其實早就不是新資訊，下跌常常是資金在換股、提款或重新布局，而不是產業邏輯瞬間崩壞。

## 產業 / AI 基礎設施

- 本集核心重點放在 NVIDIA 在 CES 與後續產品規畫帶出的新方向。謝孟恭認為老黃強調的不是單一晶片，而是整個 whole rack solution，包含新晶片、線材、機櫃、散熱與系統整合，代表硬體演進還遠沒有結束。
- 他特別拆解 NVIDIA 提出的 ICMS 概念，認為這是把 AI 推論時龐大的 context / KV cache 壓力，從 HBM 與系統記憶體部分下放到更高效的 storage 節點，藉此提升整體推論效率。
- 他從 CES 現場照片延伸，注意到 NVIDIA 展示設備中出現群聯控制器，進一步推測未來 DPU 驅動的 storage server、經過認證的 SSD 與 pseudo-SLC 類型應用，可能都會成為新的投資焦點。
- 在他看來，這說明 NVIDIA 並不是等市場痛點爆發後才反應，而是很早就已在為 prefilling、de-coupling、context memory 等問題預先布局，因此供應鏈的新故事還會不斷冒出來。

## 個股 / 公司觀察

- 他提到像帽聯、台光電等高共識 AI 股出現回檔，市場會拿 cable-less design、材料規格調整等理由解釋，但他認為這些更多是跌勢中被拿來放大的故事，不一定是主因。
- 對群聯，他認為若 NVIDIA 後續 storage 架構真的走向特定認證或特定控制器方案，群聯將有機會從原本的記憶體題材，進一步升級成 AI storage 新架構受惠股。
- 對聯詠、祥碩等偏消費性或相對落後的 IC 設計股，他看法仍偏保守，認為這些公司比較仰賴整體消費性景氣回升，而目前市場主線仍明顯是 AI 相關。
- 他也分享自己對研究報告的偏好：與其看通吃型產業介紹，更重視「新應用將如何改變既有供應鏈」這類真正能帶來交易優勢的深度資訊。

## 其他 / QA 重點

- QA 中談到研究員工作與投資資訊來源時，他強調真正有價值的，不是把產業架構重新抄一遍，而是能夠指出新應用、新規格變化，將如何影響既有公司與供應鏈。
- 對記憶體行情，他提醒市場常只看現貨報價，但真正更重要的是合約價與公司實際行為，例如模組廠敢舉債買顆粒、公司敢大筆備貨，往往比網路流言更有參考價值。
- 生活題材部分，他也聊到義大利旅遊、遊戲與作品推薦，整體節奏仍維持股癌一貫的市場與生活穿插風格。

## 一句話總結

- 這一集的重點是：AI 硬體主線仍在延伸，而且已從算力、記憶體再往 storage 與整機架構擴散，真正該關注的是新架構如何重寫供應鏈，不是短線鬼故事。
