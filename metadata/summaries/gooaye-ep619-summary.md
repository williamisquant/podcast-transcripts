## 開場 / 家庭照片與成為父親後的感受

- 本集由 TravelBlue 旅遊配件業配開場，先從出國睡眠品質、飛機上怎麼撐過長途旅程切入。
- 開場後他分享第一次全家福拍攝帶來的滿足感，也特別回應老聽眾與其孩子的故事，整段充滿一種成為父親後才更能理解的柔軟。
- 謝孟恭提到，有了小孩之後，自己對很多生命、照顧與安全相關的事情感受都變得更強，也更容易被別人的家庭故事觸動。

## 市場 / 加碼節奏與風險承受

- 面對近期盤勢，他的操作心法仍是先承認市場會波動，再根據自己能承受的程度慢慢把水位加回去，而不是因為情緒一次把部位灌滿。
- 他形容自己會去收「大家短期不要的東西」，但加碼也不會太快，因為在結算、政策或事件敏感期，過度急躁很容易把自己暴露在不必要的震盪裡。
- 這種做法反映的是一種偏實戰的風險管理：不是嘴巴上說長期，而是要真的有能力撐過短期雜訊。

## AI / AWS / 雲端聯盟變化

- 產業主線之一是 AWS Reinvent 相關消息與市場傳聞，包含先前延遲、後來逐步落地的 OpenAI 與 Amazon 資金或合作脈絡。
- 他認為像 Trainium 這類 AWS 自家晶片與雲端資源的推進，本質上仍是大廠在算資本效率與平台控制權，不是單純「誰技術比較酷」的問題。
- 節目中也回頭看 Google、Avago 等先前被市場熱捧的供應鏈，提醒大家熱門敘事轉得非常快，投資人不能只在情緒最滿時才開始理解故事。

## 衛星 / 通訊供應鏈

- 本集另一條很明確的題材是衛星與通訊，尤其提到 Starlink BOM 中某些零組件的占比與相關供應鏈受惠邏輯。
- 他談到這類市場不只是在補偏鄉網路，而是可能延伸到軍事、通訊備援與新的網路使用場景，因此雖然總量未必像消費電子那麼大，但應用價值不容小看。
- 在光通與模組端，他也提及 transceiver module 這類接收與傳輸訊號的環節，強調真正值得注意的是誰卡在關鍵節點、誰只是短期跟風上漲。

## 科技擴散 / 低估新市場

- 他引用談 Uber 與創投圈的觀點，說明人們常常低估一個新市場真正展開後的規模：不是因為大家不知道它存在，而是因為很難在早期就想像它長大後會怎麼重塑日常。
- 這種思路也被他拿來看 AI、衛星、雲端平台等新題材——真正大的機會，往往不是一開始最直觀的那個產品，而是之後被帶出來的延伸市場。

## 價值觀 / 善意、捐助與別把世界讓給爛人

- 後半段他花了相當多篇幅談人與人之間的善意，像是陌生人求助、社會互信下降、大家愈來愈怕被騙等現象。
- 謝孟恭的立場很清楚：與其整天喊 raise awareness，不如真的去做事、真的去捐、真的在自己有能力時幫助別人。
- 他分享自己不只捐給組織，平常生活裡遇到需要幫助的人也願意直接伸手；雖然世界確實因為詐騙與垃圾人變得更難信任，但不能因此把所有善意都關掉。
- 在親子題上，他也提到像聖誕老人這種想像，未必要急著把孩子拉回冷冰冰的現實；很多時候保留一點夢幻，本身就是成長的一部分。

## 一句話總結

- 這集一邊談 AWS、衛星與新市場擴散，一邊談家庭與善意：投資上要敢看長線、生活上也別因為爛人太多，就把自己活成一個不敢再相信任何人的人。
