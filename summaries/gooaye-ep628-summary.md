# Gooaye 股癌 EP628 | 🍑 摘要

## 提到的股票 / 公司
- 台積電
- 世界先進

## 1. 寒假對家長不是休息，而是行程管理
- 他很直白地說，小孩放假後不是不能愛，而是高密度相處本來就會累。
- 所以與其待在家裡互磨，不如提早把返鄉、出遊、活動先排好。
- 育兒在他眼裡不是純犧牲，也會逼大人把生活半徑打開。

## 2. 孩子還會主動找你的時候，能去就去
- 他分享孩子表演時會一直找爸媽位置，這讓他很有感。
- 因為這種「被孩子需要」的階段很短，晚一點就可能變成爸媽不要來。
- 所以他近年更重視身教、陪伴與自己在孩子面前的樣子。

## 3. 關稅要看相對值，不是只看絕對數字
- Transcript 裡明確談到 15% 關稅與相關討論。
- 他的核心觀點是：如果台灣和主要競爭對手拿到差不多條件，那對過去常吃虧的台灣其實是相對利多。
- 不要拿一兩天股價漲跌，去替政策好壞下最終結論。

## 4. 台積電法說之後，重點是錢往哪裡流
- 他認為真正值得盯的是台積電 capex 外溢，而不是社群上的政治嘴砲。
- 世界先進在 8 吋相關題材上被特別提到；設備、材料、後段鏈也都在觀察名單。
- 法說後若股價先震，不代表基本面就翻空。

## 5. 設備 / 廠務是主菜，被動元件像下一道上桌的菜
- 他原本重心仍在設備與廠務，但法說後部分資金開始轉向更有動能的被動元件。
- 被動元件有機會成為下一個出事的地方，但品項間強弱差很多。
- 方向看對很重要，真正會不會賺，還是要看市場下一步準備交易哪一段。

## 一句話總結
這集把育兒、關稅、台積電 capex 與被動元件串在一起，底層只有一句：先看清楚相對位置與真正重要的事，再把時間、資金和情緒放到最值得的地方。
