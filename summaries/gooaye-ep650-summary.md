## 用投資人的方式管理身體：不只看體重，而是看結構有沒有變健康

這集開場先從第二針瘦瘦針的心得談起。
謝孟恭提到，這次食量縮小得更明顯，但沒有到讓人反胃或完全吃不下的程度；
他也不只盯著體重下降，而是會持續看體脂與肌肉量，避免最後瘦掉的其實都是肌肉。
再加上開始打網球、學泰拳，整體上就是把身體重新當成一個需要長期配置的資產來管理。

這種講法其實和他做投資非常像：不是只看表面數字有沒有變漂亮，而是看底層結構有沒有變健康。
若只是體重下降卻把身體機能搞爛，不能算成功；
同樣地，若只是帳面報酬好看卻建立在過度風險之上，也未必是真正好的投資。

## 這波先當多頭修正看，但油價若失控就要改劇本

回到市場，謝孟恭認為當時最麻煩的不是單純下跌，而是中東戰事與油價一起把不確定性大幅拉高。
不過在大框架上，他仍先把這波視為多頭中的修正，而不是立刻定義成結構性崩盤。
這個判斷很重要，但他同時也保留很大的彈性：暫時先這樣看，不代表死守這個劇本。

真正會改變他看法的，不是新聞標題有多刺激，而是油價是否一路創高、最後造成需求破壞。
只要油價沒有失控，市場仍有很高機率只是高波動修正；
但若油價持續上衝、壓縮企業與消費端，整套評價框架就必須重寫。
這再一次顯示，他做判斷時非常重視可以追蹤的變數，而不是靠情緒選立場。

## Agentic AI 讓 CPU 從配角重新走回主舞台

這集最重要的產業主題，是他正式把注意力大幅轉向 CPU。
謝孟恭指出，Agentic AI 和過去聊天型 AI 最大的差別，在於前者會有大量 reasoning、工具呼叫、API 轉換、資料解析與程式執行，這些其實都落在 CPU 更擅長的工作範圍。
節目中他甚至用一個很鮮明的比喻：GPU 像場上負責瘋狂砍分的超級球員，CPU 則更像在旁邊負責指揮、調度與串起整個工作流的教練。

這個比喻很有代表性，因為它不是在捧 CPU 踩 GPU，而是在提醒市場：AI 系統從單點運算走向複雜工作流之後，原本被當成配角的元件，可能重新變得關鍵。
謝孟恭也特別強調，不要因為某一段時間 GPU 強，就把 CPU 講得像垃圾；
現在 CPU 變重要，也不代表 GPU 沒用了。
AI 時代變化太快，真正成熟的看法是理解它們在不同階段各自扮演什麼角色。

## 研究題材不能只自己腦中自洽，還要回頭看公司派怎麼講

他在節目裡分享自己的研究方法：會先從論文、報導與產業趨勢去發想，再回頭確認管理層與公司派實際怎麼說，避免自己只是把邏輯推得很順，卻沒有任何產業端驗證。
他引用 Lisa Su、Intel 管理層以及 NVIDIA 在展會上的動作，去確認 CPU 需求確實正在升溫，尤其是在推論與 agent 工作流端更值得注意。

這個方法很重要，因為很多投資人的研究其實停在故事講得通，但謝孟恭在意的是故事要不要先過產業端這一關。
公司派不一定會把話講滿，但他們的語氣、重點與資本支出安排，常常已經足以透露方向。

## CPU 受惠鏈不能一個標題打天下，要分純度、平台位置與滲透路徑

在受惠鏈上，他直接點到 AMD、Intel、ARM，也提到 NVIDIA 的 whole rack solution 與平台整合地位。
這裡的重點不是單純列名單，而是提醒大家：同樣都叫 CPU 受惠，受惠方式其實完全不同。
有人是賣晶片本身，有人是吃平台整合，有人是透過授權與生態滲透分到一杯羹。

這種拆法和他看其他 AI 題材的方式完全一致：方向看對不夠，還要看你買到的是不是那條邏輯最直接、最有純度，或至少最容易被市場認知的標的。
否則很容易出現方向對了、位置卻買錯的情況。

## 戰爭還會帶出旁支題材，但旁支不該假裝成主線

除了 CPU 主線外，他也留意中東戰事是否可能透過鋁廠受損、鋁價上漲，進一步傳導到鋁箔、電容、被動元件等方向。
這類題材在他眼裡比較像戰爭風險延伸出來的對沖與備用題材，而不是主軸本身。
這也很能反映他的研究方式：當主風險出現時，不只思考什麼會受傷，也會順手想哪些地方可能被迫受惠。

但他也很快提醒，很多看似邏輯很順的旁支題材，如果沒有真正接上產業需求與漲價機制，就很容易只是腦中自洽。
願意去想，不代表要馬上把它當主線，這就是他的克制。

## 本集總結

EP650 的重點是：在中東風險與油價干擾下，謝孟恭仍先把市場視為多頭修正，並正式把 AI 主線從 GPU 延伸到 CPU 與相關供應鏈；
而在市場之外，他也再次提醒，健康與身體狀態，才是你真正的核心資產。
