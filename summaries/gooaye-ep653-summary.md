## 提到的股票 / 公司

- 台股：台積電、聯發科、創意
- 美股：Google、Broadcom、Marvell、Intel、AMD、NVIDIA、高通

## 1. 開場 / 生活話題

### 善變不一定是缺點
- 前段雖然有 NordVPN 業配，但真正延伸出去的生活話題，是謝孟恭重新看待自己的個性。
- 他覺得自己屬於很容易投入新興趣、理解很快、但也可能很快轉向下一個題目的人。
- 以前這種「善變」或「三分鐘熱度」可能會被當成缺點，但他現在比較把它理解成一種探索能力：願意快速嘗試、快速學習、快速切換，其實也很適合面對變化很快的市場與工作。

## 2. 總經 / 盤勢

### 多頭行情的重點是把握，不是預設崩盤
- 他認為當下市場就是非常強，美股已經從前面的不舒服狀態大復活，台股也持續由強勢標的往上推進。
- 在這種盤裡，最重要的不是先預設哪裡要崩，而是好好多頭把握，該上車的時候就要上。
- 他也提到，最近調節一些部位不一定是因為看壞，而是因為它們漲太慢；這其實反映出現在是一個機會成本很高、資金會往最強標的集中的行情。

### 台積電仍是指數核心
- 他重申台股本質上很大程度就是台積電的形狀，所以看大盤上方空間，還是得先看台積電。
- 他的看法是，只要台積電明後年的獲利預估沒有被明顯下修，上方空間都還不能輕易說沒了。

## 3. 產業 / 供應鏈

### 聯發科 / Google TPU 線是這集主軸
- 這集最重要的產業重點，是聯發科在 Google TPU 鏈上的角色，可能比很多人原本想像的更重要。
- 他的意思不是單純講一個題材，而是認為若後續頭片、量產與導入都順利，聯發科來自 AI 的貢獻有機會在未來幾年顯著提高，甚至推動市場重新評價。
- 他也用 Broadcom 對照說明：Google 自己做主晶片，Broadcom 與聯發科分別吃到不同的設計 / 代工 / 周邊功能位置，所以不是單一贏家全拿，而是整條鏈都可能有肉。

### 市場不要把新玩家加入解讀成別人就要去死
- 他提醒市場常常喜歡把產業敘事講成零和遊戲，例如 GPU 強了 ASIC 就得死，或新供應商加入後原本玩家就會被完全擠掉。
- 但他認為 AI 現在本質上還是藍海，很多玩家都有可能分到市場，而不是一定只活一個。
- 所以遇到中間的利空、延遲、雜音時，不能只看市場情緒怎麼喊，還是要回到實際需求與供應鏈位置去判斷。

## 4. AI / Token / 算力價格

### token 長期還是會走向更便宜
- 他對 token 的核心看法很明確：長期來看，token 應該還是會越來越便宜，而不是越來越貴。
- 短期價格之所以變高，主要是 AI 的爆發速度快過供應擴張，造成需求先把產能擠滿。
- 但只要建置量慢慢跟上，算力價格更像 cloud 資源，最後還是會往規模化、普及化、價格下降的方向走。

## 5. 投資心法 / Q&A

### 持股重點是抱得住，不是什麼都抓到
- 談到美股持股時，他提到自己目前比較大的部位還是在 CPU、Marvell 與相關光通鏈。
- 但他的重點不是炫耀壓了什麼，而是提醒：市場裡最難的從來不是找到題材，而是你能不能在波動、雜音和 FUD 裡繼續抱得住。
- 很多時候你知道方向是對的，也不代表過程中不會被嚇到；所以真的重要的是維持清醒，不要因為市場一時的情緒就把自己洗出去。

## 一句話總結
EP653 的主軸很清楚：在強勢多頭的背景下，謝孟恭一邊看好聯發科 / Google TPU 這條 AI 供應鏈的後續想像，一邊提醒大家，無論是市場敘事、token 價格還是個人性格，都不要太快用單一路線把事情講死，真正重要的是保有判斷力與彈性。
