## 生活 / 健康
- 他把健康管理和投資管理放在一起看：兩者本質上都在處理複利。
- 如果天天研究報酬率，卻不敢看自己的睡眠、體脂或體檢數字，本質上也是在逃避現實。
- 平常不管，等爆掉再補救，通常都很貴。

## 市場 / 盤勢
- 他認為當時美股比較像全面性的估值壓縮，不是市場很精準地在淘汰爛公司。
- 連 Microsoft、Google 這些大公司都一起修正，表示主因更像風險偏好下降。
- 所以投資人的工作不是跟著尖叫，而是分辨哪些是被整包錯殺、哪些是真的會被新競爭結構擠壓。

## AI / Agent 與入口
- 他認為很多人把 agent 想得太神祕，但通知整理、摘要、跨 App 串接，本來就很接近 agent 的雛形。
- Apple 不一定要做最強模型；只要把入口、隱私與裝置整合做得夠好，就可能成為主要受益者。
- 這背後的重點是：入口與整合，常常比單點能力更值錢。

## 產業 / 估值理解
- 他提醒大家，市占率下滑不等於一定賺更少；如果整體市場同時擴大，絕對金額可能反而更大。
- 這種看法適用於模型公司，也適用於設備、平台與供應鏈。
- 不要迷信唯一贏家敘事，要看的是餅有沒有變大、公司能不能繼續分到夠大的那塊。

## 部位 / 心法
- 主倉仍然集中，但會保留小部位追蹤倉，讓自己不和市場脫節。
- 小倉位的用途不是賺大錢，而是維持感知與研究敏感度。
- 集中與分散不是二選一，而是怎麼分工。

## 一句話總結
這集在講同一件事：無論是健康、AI 還是部位管理，真正重要的都不是嘴上相信，而是長期可執行的紀律。
