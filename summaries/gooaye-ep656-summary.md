## 1. 開場 / 生活話題

### 紅酒與好市多
- 謝孟恭前段花不少時間聊紅酒，重點不是炫耀收藏，而是談「如何快速建立品味與辨識力」。
- 他認為一開始可以先透過名莊、特殊年份、價格帶去建立座標，喝過一輪後，才會慢慢知道自己真正喜歡哪種產區、葡萄與風格。
- 他也分享最近去好市多看酒，意外發現有些平常會跟酒商買的東西，現場價格反而更好，讓他對這類通路改觀。

### 生活調整
- 最近除了紅酒，他也持續做生活重整，包括減肥、運動與調整作息。
- 這些改變某種程度上也影響到他看盤、安排時間與跟家人的相處方式。

## 2. 市場 / CPU 與 AI 供應鏈

### CPU 類股
- 他認為目前美股依然強，CPU 類股尤其明顯，Intel 數字不差、漲價預期也讓整條線更熱。
- 他強調自己先前對 CPU 的看法不是事後收割，而是從 Agentic AI 需要更多 CPU 資源這件事一路推過來。
- 但他也提醒，市場在題材發酵後，開始出現很多過度樂觀的小作文，例如把 CPU 在 server rack 裡的配置講得太誇張。
- 他的意思不是看空 CPU，而是：長線趨勢可以看好，但短線追價時不要把最誇張的敘事也一起照單全收。

### TPU / 聯發科
- 他提到 TPU 8T、8I 這條線最近也更明朗，市場開始更清楚聯發科與 Broadcom 在裡面的角色。
- 他也承認自己先前對 training / inference 的分工有些誤判，後來才修正認知。
- 整體來看，他認為這會讓聯發科後續表現更值得關注。

## 3. 台積電 / 資金配置

### 核心看法
- 本集最重要的市場論點其實是台積電。
- 謝孟恭提到自己最近大幅調整持股，核心原因就是要騰錢買台積電。

### 背後邏輯
- 金管會調整投信持股上限，讓很多原本受限制的資金，之後有機會更直接地加碼台積電。
- 他認為這等於某種程度上「解開台積電的封印」，因為過去很多資金想多買也買不夠，現在制度鬆開，後續的補部位需求可能不小。
- 更重要的是，基金經理人本來就不敢讓績效脫鉤大盤太遠，而台積電又是最容易貼近指數的配置，所以制度一放寬，很容易形成強烈的加碼動機。
- 他也提醒，若大量資金往台積電集中，其他中小型股可能反而會被抽血，這會改變整個台股輪動的節奏。

### 做就對了

## 5. 價值觀 / 家庭

### 對社會案件的看法
- 談到社會案件時，他的情緒還是很強，尤其涉及小孩時更容易被影響。
- 但他的立場不是單純喊重刑，而是認為：可以相信制度，但不能因為要尊重制度，就容忍明顯失格、失去同理心的發言。

### 孩子、想像力與兩人世界
- 他提到陪小孩看《Bluey》時，會覺得孩子保有一點想像力其實很好，不用太早把世界全都去魅。
- 他認為人大了也一樣需要某種 vision，需要一些願意相信、願意追的東西。
- 至於夫妻相處，他認為有小孩後兩人世界一定會變少，但還是要刻意經營；有些相處不是自然會發生，而是要主動留時間給彼此。
- 他最後的態度很明確：兩人世界很好玩，但有孩子後的生活也會變得更豐富；如果現在還沒生小孩，就好好珍惜眼前只有兩個人的時光。
