## 生活 / 品味與節奏
- 他用紅酒聊了一件事：建立品味最有效的方法，常常是先有一套座標，再慢慢知道自己真正喜歡什麼。
- 這種做法其實很像研究股票：先建立框架，再從框架裡修正偏好與判斷。
- 最近他也持續在做生活重整，包括減重、運動與作息調整。

## 市場 / CPU 題材延續
- 他認為美股仍然偏強，CPU 類股尤其明顯，Intel 數字與漲價預期都讓整條線更熱。
- 不過他也提醒，題材熱起來後，市場容易出現過度樂觀的小作文，把 CPU 在整個 server rack 裡的角色講得太誇張。
- 長線方向可以看好，但短線追價不要把最誇張的敘事一起照單全收。

## TPU / 台積電與資金配置
- TPU 新一代路線變得更清楚後，他開始更重視聯發科與 Broadcom 在裡面的角色。
- 另一個更大的市場論點是台積電：他近期大幅調整持股，核心原因就是想騰錢買台積電。
- 背後邏輯是投信持股上限調整後，很多原本受限制的資金有機會更直接加碼台積電，等於某種程度上「解開封印」。
- 但這也可能造成其他中小型股被抽血，改變台股輪動節奏。

## 心法 / 股品與實戰
- 他談「股品好，績效自然好」，重點不是道德，而是不要把別人留在車上洗碗。
- 可以分享標的，但不要狂推別人進場後自己先跑。
- 很多投資經驗也真的只能靠自己下場累積，股災中的體感、進出節奏與部位拿捏，不是一直問就會內化。

## 生活 / 家庭
- 談到社會案件，他情緒仍很強，尤其牽涉小孩時更容易被影響。
- 陪孩子看《Bluey》時，他也提到人長大後依然需要某種 vision，不必太早把世界完全去魅。
- 至於夫妻相處，他認為有小孩後兩人世界一定變少，但還是要刻意經營，因為很多關係不是自然會發生，而是要主動留時間。
