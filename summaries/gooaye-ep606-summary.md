# Gooaye 股癌 EP606 摘要

## 提到的股票 / 公司
- Amazon
- Google
- Astera Labs
- BizLink

## 生活近況與心境
- 開頭先聊換季、過敏、校外教學，也順便談自己想成為什麼樣的爸爸。
- 他拿今年和去年的校外教學心情對照：同樣在陪孩子，但今年比較能待在當下，不再一直被持股波動綁架。
- 這段其實是在講成熟：不是不在乎市場，而是情緒控制比以前更好了。

## AWS 供應鏈的雜音
- 本集主線是 AWS 相關供應鏈。
- 他認為這條線前陣子被各種傳聞與空方敘事反覆放雷，但需求面沒有想像中差。
- 當原本人人看衰的鏈條開始止穩、甚至個別公司先走強，就表示市場結構可能在改變。

## 回到 Amazon 財報看 AI 基建
- 與其被供應鏈 rumor 帶著跑，他更主張直接看 Amazon / AWS 的資本支出與電力擴張。
- 他特別強調：AI 時代的資料中心競賽，重點已不只是幾台伺服器，而是 **多少電、多少地、多少 capacity**。
- 如果電力和資料中心還在持續加碼，代表需求沒有真的轉弱。

## Google 鏈與 AWS 鏈的資金切換
- 他提到前一段時間市場偏愛 Google 供應鏈，但 AWS 這邊也可能出現跌深後的再評價。
- 對長線投資人來說，重點不是每天被雜訊洗，而是分辨這是短線錯殺，還是邏輯真的壞掉。
- Astera Labs、BizLink 被他拿來當這種觀察的代表案例。

## 人會被回力鏢打到
- 後段 QA 延伸到人生位置變化：當你從員工變老闆、從單身變父母，很多以前講過的話都會被現實回打。
- 他不把這件事看成虛偽，而是成長後本來就應該修正觀點。
- 這和投資其實同一套邏輯：不是固守舊答案，而是隨現實更新。

## 一句話總結
EP606 用 AWS / Amazon 的資本支出與電力擴張，對抗市場上的供應鏈雜音；在人生面則提醒，真正成熟是你能比去年更穩。
