# Gooaye 股癌 EP631 | 🎁 摘要

## 提到的股票 / 公司
- Amazon
- ASTS

## 1. 顧小孩讓他更接受先天差異這件事
- 小孩生病時，Lisa 幾乎每小時醒來量體溫，恢復力還是超強，讓他非常佩服。
- 這也讓他更接受：很多人體質、睡眠耐受度、恢復速度，先天就真的不同。
- 後天努力重要，但不要假裝每個人都活在一樣的起跑線。

## 2. Alex Honnold 這種人，本來就不是標準模板
- 他拿攀岩、極端風險承擔者當例子，強調世界需要這種不按 NPC 劇本走的人。
- 不是每個人都要變英雄，但也不用急著用保守框架規定別人只能怎麼活。
- 這種對差異的尊重，也貫穿他對市場的看法。

## 3. 台股很瘋，但你的體感取決於部位結構
- 很多人一個月就賺贏去年，但他因為美股部位較大，體感沒有那麼極端。
- 這提醒一件很實際的事：市場很熱，不代表每個人都同步受惠。
- 先看自己到底站在哪裡，再決定如何調整。

## 4. 關稅、ownership 與網通成本轉嫁
- 他再次強調關稅要看現實力量分配，不是看情緒喊話。
- 另外也提到網通廠在原物料與記憶體壓力下，未必沒有轉嫁能力。
- 所以不能只因為「成本上升」就直接判死刑，還要拆客戶結構與報價模式。

## 5. 低軌衛星不是單一故事，而是多玩家一起放大
- Transcript 裡明確提到 ASTS 與 Amazon 的低軌衛星路線。
- 他認為當更多玩家一起進場，整條供應鏈會從題材變成真的缺貨、真的擴產、真的上修。
- 做法上還是那一套：先確認方向，再往外層挖還沒完全定價的供應商。

## 一句話總結
這集從照顧小孩、先天差異一路談到關稅與低軌衛星，核心其實很單純：承認差異、尊重權重，然後把自己放到最對的位置上。
