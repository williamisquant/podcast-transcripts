# Gooaye 股癌 EP650 | 🦄 摘要

- 節目：股癌
- 主持人：謝孟恭
- 集數：EP650 | 🦄
- 日期：2026-04-04
- 來源：clean transcript 摘要版

## 1. 開場 / 生活話題

- 開場是 KINYO 捕蚊燈業配，風格一樣用投資語言包裝生活用品。
- 他先更新自己打「瘦瘦針」第二針後的感受：食量明顯縮小，但還沒有到反胃或厭食的程度，整體算是能接受。
- 他也提到自己會持續追蹤體脂與肌肉量，重點不是只瘦體重，而是避免把肌肉掉光。

## 2. 總經 / 盤勢

### 核心觀點
謝孟恭認為當時市場最麻煩的不是單純跌，而是中東戰事與油價把不確定性拉高；但在大框架上，他仍把這波視為多頭中的修正，而不是結構性崩盤。

### 重點
- 他認為美股比台股更難做，主因在於戰事、油價與風險偏好反覆擾動。
- 從他的角度，美國先處理伊朗風險，雖然短線造成油價與市場壓力，但長期反而是在排除更大的不確定性。
- 真正要改變看法的關鍵不是新聞標題，而是油價是否一路創高並造成需求破壞。

### 操作心法
- 當市場還在高波動與高風險狀態時，不需要過度預設單一劇本。
- 可以先把這種下跌視為多頭修正，但若油價失控、盤勢持續破底，就要調整策略。
- 他反覆強調，投資不是猜故事本身，而是看市場何時開始真的反映那個故事。

## 3. AI / CPU / 產業主線

### Agentic AI 帶動 CPU 重新被看見
- 這集最重要的產業主題，是他正式把投資目光大幅移向 CPU。
- 他認為 Agentic AI 與傳統聊天型 AI 最大差別，在於前者會出現大量 reasoning、工具呼叫、API 轉換、資料解析與程式執行，這些都會大幅提高 CPU 在整體工作流中的占比。
- 在這個脈絡下，GPU 仍然重要，但 CPU 已經不只是配角，而是重新被市場定價的核心元件。

### 為什麼要先看公司派與產業端
- 他分享自己的研究方法：先從論文、報導與產業趨勢發想，再回頭看公司派怎麼講，避免自己只是把邏輯自洽後硬套到股票上。
- 他引用 Lisa Su、Intel CFO、Jensen 等人的說法，確認 CPU 需求確實在升溫，尤其伺服器與 AI 工作流端更值得注意。
- 他的核心判斷是：CPU 趨勢方向大致正確，但最後受惠幅度、誰吃到最多，仍要看產能、設計、市占與供應鏈配置。

## 4. 產業 / 供應鏈

### CPU 受惠鏈
- 他直接點名 AMD、Intel 是最容易理解的 CPU 純度標的。
- NVIDIA 雖然還是主要被當作 GPU 公司看，但因為 Vera Rubin 平台也帶有 CPU 與整體系統方案，他認為 NVIDIA 某種程度上也可以視為 CPU 題材的延伸受益者。
- ARM 也是值得注意的方向，因為在雲端伺服器與 hyperscaler 自研 CPU 裡，ARM 的存在感已越來越高。

### 鋁、鋁箔與被動元件
- 由於中東戰事波及鋁廠，他開始留意鋁價上漲是否會往鋁箔、電容、被動元件等方向傳導。
- 他的想法不是把這當成絕對主線，而是當成一種戰爭風險延伸出來的對沖與備用題材。
- 但他也提醒，很多看起來邏輯很順的題材，若沒有真的接上產業實際需求與漲價機制，就很容易只是腦中自洽。

## 5. 個股 / 公司觀察

### AMD / Intel / NVIDIA / ARM
- 他在美股端開始更積極研究 CPU 族群，認為最值得優先觀察的是 AMD 與 Intel。
- NVIDIA 不見得是 CPU 題材最純的標的，但若市場開始重估 CPU 在 AI 裡的角色，整個模型與平台價值也可能同步被上修。
- ARM 的受惠方式則比較偏向授權、生態滲透與 CSP 自研 CPU 的擴張。

### 題材與股價反應不一定同步
- 他提醒大家：即便你方向看對，也不代表股價立刻會動。
- 很多題材是太早看到、太早佈局，結果最後沒等到市場交會點；因此停損、調整與節奏感仍然很重要。

## 6. 健康 / 運動 / 生活方法

### 健康是核心資產
- 他談到自己開始打網球、持續運動、也因減重而更有意識地照顧身體。
- 一個很明確的價值觀是：工作再怎麼重要，若健康沒顧好，對生活造成的影響會比工作損失更大。
- 他把打網球與學泰拳都視為重新調整節奏、重新感受身體的一種方式。

### 三分鐘熱度不一定是缺點
- 他承認自己對很多興趣都是快速投入、快速買裝備、快速學，未必能維持很久。
- 但他並不完全把這件事看成缺點，反而把它視為探索生活、增加感受與累積不同經驗的方法。

## 7. 其他 / 聽眾互動

- QA 段落談到綽號、家庭、育兒與陪伴，氣氛偏輕鬆但也帶有不少生活體會。
- 他分享自己學生時代也有被亂取綽號的經驗，認為面對這種事，有時越有反應越容易被針對。
- 整體上這集雖然主軸仍是市場與產業，但生活感與身體感受也非常重。

## 一句話總結

EP650 的重點是：在中東風險與油價干擾下，謝孟恭仍把市場視為多頭修正，並正式把 AI 主線從 GPU 延伸到 CPU 與相關供應鏈；同時他也提醒大家，無論投資多忙，健康、運動與身體狀態才是你真正的核心資產。
