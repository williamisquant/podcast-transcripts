# Gooaye 股癌 EP621 | 🎄 摘要

## 提到的股票 / 公司
- Tesla
- AP Memory

## 1. 聖誕夜最重的一段，是孩子問爸爸會不會死
- 兒子睡前追問年紀、壽命，最後直接哭著說不要爸爸死掉，主委自己也被打到。
- 這讓整集多了一層很強的生命感：很多投資焦慮，放到家庭面前都會重新排序。
- 本集不只是市場回顧，更像一次價值觀整理。

## 2. 教養上他傾向開放，但要陪著理解
- 他不排斥孩子接觸 AI、電腦與新工具。
- 重點不是全面禁止，而是適量、陪伴、引導，不把孩子的好奇心直接堵死。
- 這跟他看新技術的方式其實一樣：先理解，再管理，不是假裝不存在。

## 3. 盤勢轉暖，但倉位管理還是第一位
- Tesla 與一些原本偏悶的部位開始動，代表資金輪動比想像快。
- 但他沒有因此放掉風控：最重要的不是把槓桿拉滿，而是不要進入被追繳保證金的局。
- 能活過波動，比短期多賺幾趴重要。

## 4. 熱門題材最後還是要回到採用率
- 他拿 immersion cooling 當例子：概念很熱，但若採用進度不如預期，就該誠實調整。
- Transcript 也提到 WOW、AP Memory 等記憶體新技術；他的態度是可以追，但不能亂把概念混成同一個故事。
- 題材可以先炒，研究不能跟著失真。

## 5. 不用把別人的配置當自己的人生標準
- 後段從轉職、竹北樣板人生聊到比較焦慮。
- 他的觀點很穩：不要讓社群上的標準家庭 / 高收入模板，偷偷變成自己的壓力來源。
- 真正重要的是，那是不是你願意承擔、也過得下去的配置。

## 一句話總結
這集把親子對話、開放式教養、記憶體新題材與倉位管理放在一起，底層其實都在講同一件事：先知道什麼才是你人生裡不能讓位的核心。
