# Gooaye 股癌 EP651 | One for all 摘要

- 節目：股癌
- 主持人：謝孟恭
- 集數：EP651 | One for all
- 日期：2026-04-08
- 來源：clean transcript 摘要版

## 1. 開場 / 生活話題

- 開場是 Saily eSIM 業配，重點放在出國上網、VPN、跨區切換與旅遊時降低不確定性。
- 謝孟恭前段花不少篇幅分享去參加林哲瑄引退儀式的經驗，覺得那是自己近年最難忘的一晚，也再次感受到節目真的有幫到一些人。
- 他也談到孩子抓周、自己其實很希望兒子未來能成為運動員，反映出他對運動員承受壓力與被全場加油那種「很帥」狀態的嚮往。

## 2. 人生觀 / 價值觀

### 被肯定與被需要
- 這集一個很重要的主軸，是他談自己其實不是一個很有自信的人。
- 他說自己從小較少得到強烈肯定，但太太、岳母、以及兩個孩子給了他非常多信心與情感上的支撐。
- 特別是大兒子會一直傳語音催他回家，讓他深刻感受到「被需要」這件事的重量，也因此越來越少出席夜間社交場合。

### 節目的意義
- 參加林哲瑄引退儀式後，他第一次更明確感受到，自己做節目不只是分享投資，也真的會在別人的人生某些時刻產生影響。
- 他不把這當成什麼偉大成就，而是把它視為一種很珍貴的回饋與緣分。

## 3. 總經 / 盤勢

### 核心觀點
謝孟恭認為台股依然偏向多頭修正後再上，美股則還不能完全鬆懈；真正要觀察的核心變數仍是中東局勢與油價。

### 重點
- 台股已重新走強，他較傾向視為多頭修正後續攻。
- 美股雖然反彈很強，甚至強到他笑稱如果是死貓跳也跳太高，但還不能百分之百確認風險解除。
- 在他的框架裡，只要油價沒有一路飆向 120、130 美元以上，市場大致還能回到常態風險評估；若油價失控，就要明顯提高警戒。

### 操作心法
- 不要在多頭市場裡輕易把部位清空。
- 真正重要的是部位水位與節奏調整，而不是 all in / all out。
- 如果技術面站回關鍵均線、修正後又守住，那戰爭雜訊對盤勢的影響就會慢慢鈍化。

## 4. 產業 / 供應鏈

### CPU / Agentic AI
- 他延續上一集的主題，認為 Agentic AI 會顯著提高 CPU 在工作流裡的重要性。
- 不只傳統 x86 CPU，連 CSP 自研 CPU 與部分 ASIC 廠自帶 CPU 的方向都值得注意。
- 但他也提醒，產業界回饋是 CPU 需求確實在升溫，卻未必像市場想像那樣會立刻帶來超大漲價與超高漲幅，還是要看訂單與產能落地。

### 記憶體與被動元件
- 記憶體產業面仍然偏缺，股票休息不等於基本面轉弱。
- 被動元件方面，鉭電容、鋁電容等仍處於偏缺狀態，有機會在後續某個事件催化下變成新族群。
- 他提醒，這條線比較像景氣循環與題材共振的類型，真正爆發往往需要一個明確 trigger。

### 設備 / 廠務 / 軍火庫思維
- 他認為如果要抓 AI 擴產大方向，仍要沿著設備、廠務、上游供應鏈去看。
- 因為不管是誰贏、哪條路線勝出，最後很多錢都得先流到建廠、製造、設備與軍火庫端。

## 5. 個股 / 公司觀察

### Intel / Tesla TerraFab
- 他提到 Intel 與馬斯克 TerraFab 計畫之間的合作訊號，認為對 Intel、Samsung 偏正面，對 Tesla 本身則較中性。
- 理由是：對 Intel 來說，這可能帶來技術授權、顧問、封裝與後段服務等高毛利機會；但對 Tesla 而言，自建晶圓與製造體系太燒錢，短期不會直接解決核心疑慮。

### Apple / 手機供應鏈
- 他觀察到部分手機品牌正在大幅下修訂單，尤其中國手機廠壓力很大。
- 相較之下，Apple 有能力吃下漲價、擴大市占，甚至在總量不佳的市場裡逆勢維持相對穩定。
- 這意味著低階或平價消費性元件供應鏈風險偏高，短線不適合硬接。

## 6. 其他 / 親子與家庭

- 抓周段落裡，他把兒子抓到的筆電、法槌、鋼琴都解讀成某種「很像自己」的象徵，但也坦白說，其實不希望孩子只是複製自己的人生。
- 他更期待孩子能找到真正喜歡、甚至比自己更熱血的人生方向。
- 這集的親子內容很多，整體氣氛比平常更柔軟，也更直接把家庭視為他現在最優先的核心。

## 一句話總結

EP651 的重點是：在台股偏多、美股仍看油價與中東風險的框架下，謝孟恭一邊延續 CPU、設備與供應鏈的觀察，一邊透過林哲瑄引退儀式、孩子與家庭的互動，談到節目的價值、被肯定的需要，以及自己真正越來越在乎的是家人與被需要的感覺。
