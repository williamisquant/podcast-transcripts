## 1. 開場 / 生活話題

- 開場是 MIHONG 瑪卡業配，主軸放在把體力當成投資腦力與工作效率的底層資產。
- 謝孟恭分享自己過去重度抽菸、後來戒菸的經驗，認為很多人迷戀抽菸，不只是尼古丁本身，也包含「暫時離開人群、到戶外深呼吸」帶來的放鬆感。
- 他也延伸談到新世代年輕人對抽菸、喝酒、工作服從與社交禮俗的態度更鬆動，反而更接近他心中真正做自己、重視生活品質的樣子。

## 2. 總經 / 盤勢

### 核心觀點
謝孟恭認為當下台股已明顯創高、美股也重新回到多頭結構附近；重點不是預設崩盤，而是觀察多頭是否由更多族群輪動接棒。

### 重點
- 台股已在高檔續強，他直接把這種表現視為明確的多頭。
- 美股雖然仍有壓力位，但已從過去幾個月「只有少數題材能做」的狀態，開始出現更多可用族群。
- 他特別強調，健康的多頭不是只有單一族群撐盤，而是越來越多族群輪動往上。

## 3. AI / CPU / 雲端算力

### CPU 角色被重新定價
- 他這集最核心的產業判斷，是 Agentic AI 時代下 CPU 的重要性正在被市場重新認識。
- 他的比喻是：GPU 像場上負責得分的明星球員，CPU 則像負責調度、執行工作流、串接 API、處理 in/out 的教練。
- 因為 AI 應用開始從單純聊天，轉向 reasoning、程式執行、資料查詢、工具呼叫，CPU 用量與價值都被拉高。

### AWS / 自研 CPU 線
- 他提到 AWS 連自家 Graviton 算力都顯得吃緊，甚至有客戶想包量卻拿不到，這是重要訊號。
- 這代表未來除了 GPU / ASIC 之外，CPU 也可能成為供給瓶頸，值得沿著 AWS supply chain、Intel、AMD、自研 CPU 生態繼續追蹤。
- 他認為比較保守的做法仍是先看 server 端 CPU，再觀察是否外溢到 consumer 端。

## 4. AI 軟體 / 模型觀察

### Anthropic / Mythos 論述
- 他對 Anthropic 新模型與宣傳口徑抱持保留態度，認為公司確實產品強，但有些說法帶有過度渲染成分。
- 他不完全相信「因為太強大所以不先全面開放」這套說法，更傾向認為背後也有算力不足、上線後怕被抱怨效能下滑的現實限制。
- 延伸到整體 AI 應用，他認為現在模型能力很強沒錯，但市場仍需回到供給、算力、落地體驗與商業化速度來看，而不是只聽敘事。

## 5. 產業 / 供應鏈

### 被動元件
- 他提到被動元件已開始出現缺貨與漲價跡象，尤其高階 AI 應用相關規格值得注意。
- 觀察重點包含國巨法說、KEMET 漲價，以及 MLCC、鋁電、鉭電等在 AI 伺服器上的需求變化。
- 他的判斷是：這條線現在還不像光通那樣已成顯學，但有機會成為後續輪動題材。

### 光通外溢到 SMT / 代工產能
- 他觀察到近期不只純光通廠受惠，連一些原本看似不相干、但有 SMT 產線與廠房的台股公司，也開始接到光通相關客戶外溢的需求。
- 這種模式像是客戶自帶設備、技術或訂單，把產能塞進既有廠房，讓一些原本產能閒置或低毛利的公司突然有新故事。
- 但他也提醒這屬於較深水、偏事件驅動的題材，若驗證沒過或訂單抽走，股價也可能快速反轉。

## 6. 個股 / 公司觀察

### 美股觀察方向
- 他延續看好 CPU 相關標的，特別是 Intel、AMD 與 AWS / 自研 CPU 生態系。
- 對軟體股則偏中長期持有，但短線對 Anthropic 帶來的市場情緒擾動感到煩躁。

### 台股觀察方向
- 台股整體仍強於美股，操作上更重視族群輪動，而不是硬去撿弱勢股。
- 如果資金開始集中往特定缺貨題材推進，被動元件與部分光通外圍供應鏈都可能有後續表現。

## 7. 人生觀 / 價值觀

- 他把戒菸經驗延伸成一個更大的想法：很多讓人上癮的行為，背後其實綁著某種情緒調節需求，重點不是完全否認欲望，而是理解欲望的來源。
- 對他來說，人生可以有喜歡的事、娛樂、甚至某些小上癮，但不能讓那些東西反過來變成主宰自己的 boss。
- 他也欣賞新世代對工作、生活、自我界線更清楚的態度，認為未來在 AI 讓生產力更便宜的時代，這種生活觀可能更合理。

## 一句話總結

EP652 的重點是：在台股強勢、美股逐漸擴散的多頭環境裡，謝孟恭把市場焦點從 GPU 進一步推向 CPU 與 AI 基礎設施，同時也用戒菸與世代差異提醒大家，真正重要的不只是效率與賺錢，還有你怎麼管理自己的身心、欲望與生活節奏。
